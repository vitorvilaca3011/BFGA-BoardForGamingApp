<objective>
Wire the local overlay state through `BoardView`, `BoardViewport`, and `BoardCanvas` so the renderer can draw local laser and ping visuals with the same fade-loop infrastructure used by remote trails.

Purpose: Phase 3 local gesture code needs a visual path independent of network state. This plan creates the styled-property plumbing and render-thread snapshot points for those local overlays.

Output: new `LocalLaser` and `LocalPing` styled properties on all canvas layers, timer lifecycle updated for local visibility, and render pipeline calls for local dot/trail/ping.
</objective>
