## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0A0A0A | Window base, inactive canvas surround, non-interactive negative space |
| Secondary (30%) | #111111 | Floating toolbar, bottom bar, settings/property panels, elevated chrome |
| Accent (10%) | #FF4D6D | Local laser head dot, trail, ping ring, active crosshair emphasis |
| Destructive | #C42B1C | Close/dismiss destructive actions only |

Accent reserved for: local laser head dot, active trail segments, ping ring + ping center dot, laser-specific status emphasis when needed. Never use accent for every toolbar icon, panel border, or generic hover state.

Existing theme neutrals remain unchanged: `TextPrimary #FAFAFA`, `TextSecondary #B0B0B0`, `BorderDefault #2A2A2A`, `BgOverlay #161616`, `AccentWhite #FFFFFF`.

## Visual Hierarchy And Interaction States

Primary focal point: live laser head dot on canvas at current pointer position.

Secondary focal point: fading trail behind active pointer path. Trail must visually decay from head to tail over ~1-2 seconds and never overpower board content.

Tertiary focal point: quick-tap ping marker. Ping uses expanding ring plus center dot, fades within ~2-3 seconds, and reads as short attention cue rather than persistent callout.

Interaction states:
- Laser tool selected: toolbar laser button uses existing active treatment; canvas cursor switches to crosshair.
- Pointer pressed: show laser head immediately at scene-correct position.
- Pointer move while pressed: append trail points in scene space; keep on-screen thickness and dot size visually constant across zoom.
- Quick tap: emit single ping at release location when press duration is under 200ms and travel is under 5px.
- Release, pointer leave, capture loss, or tool switch: stop emission immediately and allow final fade-out only.
- Icon-only affordance fallback: laser toolbar button keeps tooltip label `Laser Pointer`; no unlabeled icon-only action without tooltip.

Scale contract:
- Laser dot radius: 5px on screen.
- Trail thickness: 3px on screen at head, fading by alpha only in this phase.
- Ping center dot radius: 4px on screen.
- Ping ring: expand from 10px to 24px on screen during fade.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Select Laser Pointer |
| Empty state heading | Laser pointer ready |
| Empty state body | Select Laser Pointer, then press and hold on board to point something out. |
| Error state | Laser pointer stopped unexpectedly. Release pointer, reselect Laser Pointer if needed, then try again. |
| Destructive confirmation | none: no destructive action in this phase |

Tooltip and helper text rules:
- Toolbar tooltip label stays `Laser Pointer` until keyboard shortcut ships in later phase.
- Any transient status copy should describe gesture plainly: `Press and hold to point` or `Tap to ping`.
- Do not use generic labels like `OK`, `Submit`, or `Activate`.
