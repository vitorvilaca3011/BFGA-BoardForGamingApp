<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| overlay state -> render thread | transient laser state crosses from UI-thread properties into custom draw operation |
| remote laser churn -> render invalidation | high-frequency updates could force expensive full redraws |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-05 | Denial of Service | remote laser render path | mitigate | dedicated `LaserOverlayCanvas` owns laser timer/invalidation so remote laser churn no longer calls `BoardCanvas.InvalidateVisual()` |
| T-04-06 | Tampering | overlay render snapshot | mitigate | snapshot overlay state in `LaserOverlayDrawOperation` constructor on UI thread before render-thread use |
</threat_model>
